# Online Store Application - Kaspi.kz Catalog

Этот проект является частью Лабораторной работы №4 по дисциплине Web Development. Это веб-приложение на Angular, которое отображает каталог товаров, имитируя функционал магазина Kaspi.kz.

## Функциональные возможности

В соответствии с требованиями лабораторной работы реализовано:
- Интерактивный каталог: Отображение более 10 реальных товаров с Kaspi.kz с использованием директивы *ngFor.
- Галерея изображений: Каждая карточка товара содержит галерею из минимум 3 фото с кнопками переключения.
- Детальная информация: Вывод названия, описания, цены в KZT и визуального рейтинга товара.
- Функция Поделиться: Кнопки для быстрой отправки ссылки на товар в WhatsApp и Telegram с использованием функции encodeURIComponent().
- Адаптивный дизайн: Сетка товаров построена на CSS Grid/Flexbox и корректно отображается на разных устройствах.

## Стек технологий

- Angular (версия 17+).
- TypeScript (использование интерфейсов для моделей данных).
- CSS3 (компонентные стили).

## Структура проекта

Проект организован согласно стандартам Angular:
- src/app/models/product.model.ts - Интерфейс продукта.
- src/app/components/product-list/ - Компонент для сетки товаров.
- src/app/components/product-card/ - Компонент для отдельной карточки товара.

## Инструкция по запуску

Для локального запуска выполните следующие шаги в терминале:

1. Склонируйте репозиторий и перейдите в папку проекта.

2. Установите необходимые зависимости:
npm install

3. Запустите сервер разработки:
ng serve

4. Откройте браузер по адресу:
http://localhost:4200
